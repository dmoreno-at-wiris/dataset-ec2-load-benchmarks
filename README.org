* Benchmarks for the dataset loading to an EC2

** Description

The purpose of this project is to measure different possibilities or solutions
to load and store the ML datasets used to train a model.

*** Solutions

**** Given local storage - Dataset as it is

Executed as:

#+begin_src python
  train_local_dataset = LocalFSDataset(
        datasets_path=Path("<LOCAL_PATH_TO_DATASETS>"),
        data_annotation_file_path=Path(
            "strokes/wiris-math-online-incomplete/train_21082019.txt"
        ),
    )

    train_local_dataloader = DataLoader(
        train_local_dataset,
        batch_size=32,
        num_workers=8,
        prefetch_factor=4,
        pin_memory=cuda.is_available(),
    )

    local_fs_benchmark = DatasetLoadBenchmark(
        load_from=DatasetSource.S3,
        load_as=DatasetMode.FILES,
        dataloader=train_local_dataloader,
        iterations_number=1,
    )

    local_fs_benchmark.measure()
#+end_src
